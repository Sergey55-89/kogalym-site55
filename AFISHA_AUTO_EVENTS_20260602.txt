Афиша — автообновление реальных событий

Что добавлено:
1. events-data.json — файл с реальными событиями, который читает страница афиши.
2. scripts/update-events.mjs — сборщик событий из открытых источников.
3. .github/workflows/update-events.yml — GitHub Actions, который обновляет events-data.json каждый день и может запускаться вручную.
4. events.html теперь не держит фейковые карточки: карточки строятся из events-data.json.
5. Фильтры, стрелки недель, модальное окно «Подробнее» работают по реальным данным.

Важно:
- Для автообновления GitHub Actions должен быть включён в репозитории.
- Первое обновление можно запустить вручную: вкладка Actions → Update Kogalym events → Run workflow.
- Если внешний сайт поменяет разметку, скрипт оставит резервные события и не сломает страницу.
