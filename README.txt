Открой Когалым — статический городской портал

Состав:
- index.html — главная
- routes.html — маршруты с переходом в Яндекс.Карты
- places.html + places.js — места, фильтр, поиск, модальные карточки, карта/маршрут
- events.html — афиша
- announcements.html + announcements.js — объявления
- admin.html + admin-patch.js — локальная админка объявлений
- style.css — общий дизайн

Проверено:
- пустые about.html и README.txt заполнены
- мёртвые пустые ссылки заменены
- JS-файлы проходят node --check
- кнопки маршрутов открывают Яндекс.Карты в новой вкладке
- карточки мест сохраняют открытие модалки, прокрутку и кнопки маршрута

Запуск:
Открыть index.html в браузере или загрузить всю папку на хостинг как статический сайт.
